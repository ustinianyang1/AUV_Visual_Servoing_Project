# 仿真参数合理范围基准 (Simulation Parameter Bounds)

每次仿真运行结束后，需根据此文件限定的物理范围校验 `data/` 下生成的变量数据是否在合理区间内。若超出此范围，应记录并汇报警告信息：

- `e_x` (X轴方向的位置误差): [-0.2, 0.2] m
- `e_y` (Y轴方向的位置误差): [-0.2, 0.2] m
- `e_z` (Z轴方向的位置误差): [-0.2, 0.2] m
- `e_psi` (偏航角误差): [-0.015, 0.015] rad
- `f_x` (X轴方向的推力): [-1.4, 1.4] N
- `f_y` (Y轴方向的推力): [-1.4, 1.4] N
- `f_z` (Z轴方向的升力/推力): [-1.4, 1.4] N
- `tau_psi` (控制偏航的扭矩): [-1.4, 1.4] N.m
- `hat_tau_D1` (X轴方向估算的干扰力): [-1.4, 1.4] N
- `hat_tau_D2` (Y轴方向估算的干扰力): [-1.4, 1.4] N
- `hat_tau_D3` (Z轴方向估算的干扰力): [-1.4, 1.4] N
- `hat_tau_D4` (偏航方向估算的干扰扭矩): [-1.4, 1.4] N.m
