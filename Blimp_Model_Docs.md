# 室内飞艇轨迹跟踪控制 - Simulink 模型解析

本模型严格按照 IEEE TIE 论文《Adaptive Output Feedback Trajectory Tracking Control of an Indoor Blimp》重构。为了确保模型在连续域仿真中不崩溃，进行了底层的工程化解耦与重组。

## 1. 核心架构拓扑
模型整体闭环分为三大子系统：
*   **Plant (本体动力学)**: 负责求解式(1)和式(2)，输出真实的 $x_1$ 和 $x_2$。内部实现了时变科里奥利矩阵 $C(v)$ 的实时动态构建。
*   **Observer (ESN 观测器)**: 解决速度 $x_2$ 不可靠的问题。内部使用了带有反馈延时的储备池网络，通过 $t_d=0.01s$ 的传输延时模块（Transport Delay）保留历史状态特性。
*   **Controller (TABLF 鲁棒控制器)**: 采用反步法设计。模型规避了对虚拟控制律解析求导产生的爆炸项，转而利用一阶近似高通滤波器提取 $\dot{v}_c$。

## 2. 论文漏洞与修复记录
在模型复现中，实施了以下关键修复（已在 MATLAB Function 中硬编码）：
1.  **公式(24)的李雅普诺夫求导漏项**：强行向 $\Gamma_{li}$ 和 $\Gamma_{hi}$ 的分母 $\cos$ 项补充了平方，防止系统越界碰撞。
2.  **公式(29)代数环与除零死锁**：在 $\Lambda_i$ 求解中引入 `sign(z1i)*1e-6` 保底极小值，解除误差趋于 0 时的除零崩溃风险。
3.  **维度一致性校验**：将输入权重矩阵 $W_{in}$ 的维度强制从 $n \times 12$ 修正为与输入特征数一致的 $n \times 16$；纠正了原文将偏航阻尼误写为横滚阻尼 $D_{\omega_x}$ 的问题。

## 3. 运行环境与求解器建议
*   **MATLAB 版本**: MATLAB R2024b 或更新版本。
*   **求解器**: `ode45 (Dormand-Prince)`。
*   **特殊配置**: 由于 $q(z_{1i})$ 在零点存在硬切换，必须在 Model Settings 中开启 **Zero-crossing detection (过零检测)**，确保求解器在飞艇误差穿过设定平面时自动缩减步长以平滑过渡。
